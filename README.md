# 🚀 LeetCode 75 Solutions

This repository contains my solutions to the **LeetCode 75** study plan using **JavaScript**.

The goal of this repository is to strengthen my understanding of **Data Structures & Algorithms (DSA)** by solving carefully selected interview questions while writing clean, readable, and efficient code.

---

## 📌 About LeetCode 75

The **LeetCode 75** study plan is a curated list of 75 coding interview problems covering the most important DSA topics frequently asked in technical interviews.

Topics include:

- Strings
- Arrays
- Two Pointers
- Sliding Window
- Prefix Sum
- Hash Map & Hash Set
- Stack
- Queue
- Linked List
- Binary Tree
- Binary Search Tree
- Graphs
- Binary Search
- Heap / Priority Queue
- Dynamic Programming
- Backtracking
- Bit Manipulation
- Intervals
